<h3>功能</h3>
<h4>帮帮忙™</h4>
<p>一个提供各种实用功能的悬浮球，可从通知栏“快速设置”图块打开。</p>
<h5>公开招募规划</h5>
<p>使用5个需求标签组合出可能的方案。</p>
<ul>
<li>多渠道不同干员进度，明日方舟（含哔哩哔哩和龍成網路客户端）、Arknights 、アークナイツ 和 명일방주 随心切换，无需更改系统语言</li>
<li>支持招募时限选择</li>
<li>触摸标题时隐藏内容，抬起重置标签</li>
</ul>
<h5>寻访计数器</h5>
<p>记录已寻访次数并获取下次获得6★概率。</p>
<h5>材料来源推荐</h5>
<p>提供养成材料的高性价比掉落关卡。</p>
<h4>戳戳乐™</h4>
<p><strong>在关于界面一直戳“版本类型”，更新为 Originium 后覆盖安装应用或重启设备即可解除封印</strong></p>
<p>可选择非剿灭作战关卡持续进行代理作战，解放双手，不用一直盯着屏幕。</p>
<ul>
<li>Android 7.0 及更高版本基于无障碍 API 实现，无需 ADB 或 Root 权限</li>
<li>不截取屏幕内容，支持多分辨率</li>
<li>可设置行动间隔和时长，自带波动间隔（±150ms）和偏移坐标（±5px）点击</li>
<li>首页长按图标可自定义坐标</li>
</ul>
